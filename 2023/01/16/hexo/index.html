How to build a personal blog website

Created2023-01-16|Updated2023-01-18|O&M

We have many ways to build a personal blog. You can use blogging platforms such as CSDN, cnblogs. You can also use an automated website builder such as WordPress. Quick Start

Install hexo

Make sure you have node.js and git environment installed

After installing Hexo, execute the following command, and Hexo will create the required files in the specified folder.

After the new folder is created, the directory of the specified folder is as follows:

_config.yml

This file contains project configuration information, where you can configure most of the parameters.

package.json

This file contains information about the project. EJS, Stylus, and Markdown renderer are installed by default and can be removed freely.

Scaffolds

Template folder. When you create a new article, Hexo builds the file based on scaffold.

Hexo‚Äôs templates refer to the content that is populated by default in the newly created article file. For example, if you modify the Front-matter content in scaffold/post.md, the change will be included every time you create a new article.

Source

The resources folder is where the user‚Äôs resources are stored. Files / folders that begin with _ (underscore) and hidden files will be ignored except for the _posts folder. Markdown and HTML files are parsed and placed in the public folder, while other files are copied.
